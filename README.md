# 🐍🎲 Snakes and Ladders Game

---

## 📌 Overview

The **Snakes and Ladders Game** is a backend simulation of the classic board game. It models players, dice rolls, snakes, ladders, and game progression using object-oriented design principles.

This project demonstrates strong **Low-Level Design (LLD)** skills, **game logic implementation**, and **state management**.

---

## 🚀 Features

* 🎲 Dice roll simulation
* 👥 Multiple players support
* 🐍 Snakes (move down)
* 🪜 Ladders (move up)
* 🔄 Turn-based gameplay
* 🏁 Win condition detection
* 🚫 Valid move constraints (no overflow beyond board size)

---

## 🧠 Design Highlights

* Object-Oriented Design (OOP)
* Modular and extensible architecture
* Clear separation of concerns
* Efficient game state handling
